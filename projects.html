Airport Project

Python, Flask, SQL, Agile, Relational Databases


A group project to create a database system to simulate the needs of an Airport. Created a terminal UI alongside a GUI made using Python Flask. The UI's could interact with the database and perform CRUD operations. Made use of AWS RDS for the database and implemented secure storage by encrypting passwords.

Webscraper Project

Python, Flask, AWS EC2, Jenkins, Vagrant, Ansible, VirtualBox


Created a web app for a Python webscraper, using Python Flask. Created a CI/CD pipeline with Jenkins to help with the development. The web app was hosted on an EC2 instance on AWS within its own VPC and subnet. The cloud infrastructure was created using Ansible. Made use of Vagrant and Oracle VM VirtualBox for the development environment.

IAC with Ansible

NodeJS, AWS EC2, Ansible, MongoDB


Deployed a Node.js app on AWS EC2 using Ansible to provision the instances. Connected the app to a mongoDB server running in a separate EC2 instance. The app would then be up and running on a public IP from the running of a single Ansible playbook.
